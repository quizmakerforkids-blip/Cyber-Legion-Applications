# Cyber Legion Short Applications With Status

This version includes:

- Shorter staff application
- Immediate submitted-entry view
- Saved application in the applicant's browser
- Status refresh button
- Pending, Under Review, Accepted, and Rejected statuses
- Admin status controls

## Required Supabase update

Open Supabase SQL Editor and run the updated `supabase.sql` once.

## Deploy

Replace the files in your GitHub repository with these files and commit them. Netlify will redeploy automatically.

Public page:
`https://YOUR-SITE.netlify.app`

Admin page:
`https://YOUR-SITE.netlify.app/admin`
